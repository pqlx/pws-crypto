# Cryptography: A Concise Overview
This repository contains all code written for our paper, "Cryptography: A Conscise Overview".

Composition:
* `math/` stores all mathematical primitives. (i.e Greatest Common Divisor, Chinese Remainder Theorem, Modular exponentiation)
* `hash/`  stores all (cryptographic) hash functions. (i.e MD5, SHA-1)
* `asymmetric/` stores all asymmetric cryptosystems, (i.e RSA)
* `symmetric/`  stores all symmetric cryposystems, (i.e AES)
